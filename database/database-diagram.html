Table: users

Stores farmer user profiles and authentication data

Column Type Constraints Description
id PK UUID PRIMARY KEY Unique user identifier
mobile_number VARCHAR(15) UNIQUE, NOT NULL User's mobile number (login credential)
name VARCHAR(100) NOT NULL Farmer's full name
language VARCHAR(10) DEFAULT 'en' Preferred language (en, hi, te)
state VARCHAR(50) - State name
district VARCHAR(50) - District name
primary_crop VARCHAR(50) - Main crop cultivated
latitude DECIMAL(10,8) - Location latitude
longitude DECIMAL(11,8) - Location longitude
is_active BOOLEAN DEFAULT true Account status

INDEX mobile_number, state+district, is_active, created_at

Table: disease_detections

Stores crop disease detection results from AI analysis

Column Type Constraints Description
id PK UUID PRIMARY KEY Detection record ID
user_id FK UUID REFERENCES users(id) User who uploaded image
image_url TEXT NOT NULL S3/Cloudinary image URL
crop_type VARCHAR(50) NOT NULL Type of crop in image
disease_name VARCHAR(100) - Detected disease name
confidence DECIMAL(5,2) 0-100 AI confidence percentage
severity VARCHAR(20) low/medium/high/critical Disease severity level
treatment_recommendations TEXT - Treatment advice

INDEX user_id, created_at, disease_id, location, crop_type

Table: advisories

Stores farming advisory queries and AI-generated responses

Column Type Constraints Description
id PK UUID PRIMARY KEY Advisory record ID
user_id FK UUID REFERENCES users(id) User who asked question
query TEXT NOT NULL User's farming question
response TEXT NOT NULL AI-generated answer
category VARCHAR(50) - Advisory category
rating INTEGER 1-5 User rating (helpful/not)

INDEX user_id, created_at, category, rating

Table: government_schemes

Master data for government agricultural schemes

Column Type Constraints Description
id PK UUID PRIMARY KEY Scheme ID
scheme_code VARCHAR(50) UNIQUE Official scheme code
name_en/hi/te TEXT NOT NULL (en) Scheme name in 3 languages
description_en/hi/te TEXT NOT NULL (en) Scheme description
category VARCHAR(50) NOT NULL Scheme category
benefits_en/hi/te TEXT[] - List of benefits
eligibility_criteria JSONB - Eligibility rules
applicable_states TEXT[] - Applicable states
applicable_crops TEXT[] - Applicable crops

INDEX category, is_active, applicable_states (GIN), applicable_crops (GIN)
