STEP 4: RUN DEPLOYMENT SCRIPT (90 min)

[ ] 4.2 Run: aws s3 cp s3://agrinext-images-1772367775698/phase2/deploy/deploy-phase2.sh . --region us-east-1

STEP 5: ADD API KEYS (10 min)
------------------------------
[ ] 5.1 Open NEW EC2 terminal tab (keep first one open!)
[ ] 5.2 Run: cd ~/agrinext-phase2/backend
[ ] 5.3 Run: nano .env
[ ] 5.4 Update TWILIO_ACCOUNT_SID with your value
[ ] 5.5 Update TWILIO_AUTH_TOKEN with your value
[ ] 5.6 Update TWILIO_PHONE_NUMBER with your value
[ ] 5.7 Update OPENAI_API_KEY with your value
[ ] 5.8 Update HUGGINGFACE_API_KEY with your value
[ ] 5.9 Update HUGGINGFACE_MODEL with your value
[ ] 5.10 Update GOOGLE_PROJECT_ID with your value
[ ] 5.11 Generate JWT secrets: openssl rand -base64 32 (run twice)
[ ] 5.12 Update JWT_SECRET with first generated secret
[ ] 5.13 Update REFRESH_TOKEN_SECRET with second generated secret
[ ] 5.14 Save file: Ctrl+X, Y, Enter
[ ] 5.15 Verify: cat .env | grep -E "TWILIO|OPENAI|HUGGING|GOOGLE|JWT_SECRET="
[ ] 5.16 Go back to FIRST terminal tab
[ ] 5.17 Press Enter to continue deployment
[ ] 5.18 Wait for deployment to complete (10-15 minutes)
